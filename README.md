# Trackstar (Android)

Kotlin + Jetpack Compose client for Trackstar, a fitness coaching platform. Sibling to the [iOS app](https://github.com/vasilisneo18/trackstar-ios), talking to the same [Spring Boot backend](https://github.com/vasilisneo18/fitness-book-api).

## Status

Just scaffolded — not yet built out. See `CLAUDE.md` for stack details and how this relates to the rest of the platform.

## Getting started

Open in Android Studio, let it sync Gradle, run on a device/emulator. Or from the CLI:

```
./gradlew assembleDebug
```
